Next we can focus on the images. The first one appears to be several ducks lined up. Along with the flavor text '...from the horses mouth...', which is an idiom referring to animals, we can infer that this is meant to represent the phrase 'Ducks in a row'

At this point, solvers might notice that the each feeder answer nearly shows up in each phrase. i.e. ducks in A ROW, nearly contains the word ARROW, missing a single letter.

Solvers are able to match each answer with a phrase, and extract the position of the missing letter. Each image also came with a word, from which we can extract a letter at the same place as the missing letter from the answer
